# Flutter UI Design Rules

Design philosophy:
- 极简 + 科技感 + 高级感
- 类似 Bloomberg / TradingView / Apple 风格
- 深色金融终端风格

Layout rules:
- 统一 8px spacing system (8 / 16 / 24 / 32)
- 页面必须对齐整齐
- 不允许组件拥挤
- 使用卡片式布局
- 保持清晰的视觉层级

Component rules:
- 优先复用已有 Flutter widgets
- 所有组件必须可复用
- 不允许创建重复组件

Typography:
- 字体层级清晰
- title / subtitle / body 必须区分
- 避免默认 Material UI 样式

UI quality rules:
- 页面必须现代化
- 保持科技感
- 不允许低级 UI
- 避免默认 Flutter demo 风格
